pmdarima.datasets.load_heartrate¶

pmdarima.datasets.load_heartrate(as_series=False, dtype=<class 'numpy.float64'>)[source][source]¶

Uniform heart-rate data.

A sample of heartrate data borrowed from an MIT database. The sample consists of 150 evenly spaced (0.5 seconds) heartrate measurements.

Parameters:

as_series : bool, optional (default=False)

Whether to return a Pandas series. If False, will return a 1d numpy array.

dtype : type, optional (default=np.float64)

The type to return for the array. Default is np.float64, which is used throughout the package as the default type.
Returns:

rslt : array-like, shape=(n_samples,)

The heartrate vector.

References

[R79]Goldberger AL, Rigney DR. Nonlinear dynamics at the bedside. In: Glass L, Hunter P, McCulloch A, eds. Theory of Heart: Biomechanics, Biophysics, and Nonlinear Dynamics of Cardiac Function. New York: Springer-Verlag, 1991, pp. 583-605.
