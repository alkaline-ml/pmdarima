pmdarima.arima.is_constant

pmdarima.arima.is_constant(x)[source][source]

Test x for constancy.

Determine whether a vector is composed of all of the same elements and nothing else.

Parameters:

x : array-like, shape=(n_samples,)

The time series vector.

Examples

>>> import numpy as np
>>> x = np.array([1, 2, 3])
>>> y = np.ones(3)
>>> [is_constant(x), is_constant(y)]
[False, True]
