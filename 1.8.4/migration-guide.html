pmdarima Migration guideÂ¶

In issue #34 we made the decision to migrate from the pyramid-arima namespace to the pmdarima namespace to avoid collisions with the web framework named pyramid.

Migration is simple:

$ pip install pmdarima

Rather that importing functions and modules from the pyramid package, simply import from pmdarima instead:

from pmdarima.arima import auto_arima

Or just import it as a namespace:

import pmdarima as pm
my_model = pm.auto_arima(my_timeseries)

For further installation instructions, check out the Setup and Quickstart guides.
